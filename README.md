# Paper Applets — Resúmenes Interactivos de Investigación

Esta carpeta contiene **resúmenes interactivos** (applets) de artículos científicos en los que he tenido participación como autor. Cada archivo HTML es una visualización autocontenida que permite explorar los resultados clave de manera dinámica.

## 🚀 Cómo usar

1. **Abrir localmente**: Haz doble clic en cualquier archivo `.html` para abrirlo en tu navegador web.
2. **Interactuar**: Cada applet incluye controles deslizantes para modificar parámetros físicos y observar cómo cambian los resultados.
3. **Explorar**: Las visualizaciones 3D rotan automáticamente y los gráficos se actualizan en tiempo real.

### Requisitos

- Navegador web moderno (Chrome, Firefox, Edge, Safari)
- Conexión a internet (para cargar las librerías Three.js y Chart.js desde CDN)

## 🎯 Propósito

Estos applets están diseñados para:

- **Divulgación científica**: Presentar resultados de investigación de manera accesible
- **Material docente**: Ilustrar conceptos de ciencia de materiales y física cuántica
- **Portafolio académico**: Mostrar contribuciones de investigación de forma interactiva
- **Discusión con colegas**: Explorar parámetros y sus efectos durante presentaciones

## 📊 Características técnicas

Cada applet incluye:

- **Visualización 3D** (Three.js) de la estructura/material estudiado
- **Gráficos interactivos** (Chart.js) de las curvas y datos principales
- **Controles de parámetros** para explorar el espacio de simulación
- **Ecuaciones clave** del modelo teórico
- **Metadatos completos**: autores, DOI, referencia bibliográfica
